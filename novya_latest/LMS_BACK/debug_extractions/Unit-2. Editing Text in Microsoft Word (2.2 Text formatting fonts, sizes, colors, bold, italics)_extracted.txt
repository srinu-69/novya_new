2.2 Text formatting: fonts, sizes, colors, bold, italics
Q1. Which option is used to change the font style in a document?
a) Font size
b) Font type
c) Bold
d) Italics
Correct answer: b) Font type
Q2. To make text bold, which shortcut key can be used?
a) Ctrl + I
b) Ctrl + B
c) Ctrl + U
d) Ctrl + S
Correct answer: b) Ctrl + B
Q3. To make text italic, which shortcut key is used?
a) Ctrl + I
b) Ctrl + B
c) Ctrl + U
d) Ctrl + A
Correct answer: a) Ctrl + I
Q4. Changing the size of the text is done using:
a) Font color
b) Font size
c) Bold

--- PAGE 2 ---
d) Italics
Correct answer: b) Font size
Q5. Which option changes the color of the text?
a) Highlight
b) Font color
c) Bold
d) Italics
Correct answer: b) Font color
Q6. Underline text can be applied using:
a) Ctrl + B
b) Ctrl + I
c) Ctrl + U
d) Ctrl + S
Correct answer: c) Ctrl + U
Q7. If you want to increase the size of a heading, you should change:
a) Font color
b) Font size
c) Bold

a) Ctrl + B then Ctrl + I
b) Ctrl + U then Ctrl + S
