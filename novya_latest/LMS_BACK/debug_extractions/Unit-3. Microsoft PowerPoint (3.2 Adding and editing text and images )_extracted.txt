3.2 Adding and editing text and images
Q1. To add text to a document or slide, you usually:
a) Click on a text box or placeholder
b) Insert → Picture
c) File → Save
d) Layout → Choose layout
Correct answer: a) Click on a text box or placeholder
Q2. To edit existing text, you should:
a) Select the text and type new content
b) Insert → Table
c) Home → Font
d) Right-click → Save
Correct answer: a) Select the text and type new content
Q3. To move an image in a document, you should:
a) Drag it to the desired position
b) Press Ctrl + C
c) Change font style
d) Press Ctrl + V
Correct answer: a) Drag it to the desired position
Q4. To resize an image, you should:
a) Drag its corner handles

--- PAGE 2 ---
b) Change the font
c) Press Ctrl + M
d) Use bullets
Correct answer: a) Drag its corner handles
Q5. Shortcut key to copy text or an image is:
a) Ctrl + C
b) Ctrl + V
c) Ctrl + X
d) Ctrl + S
Correct answer: a) Ctrl + C
Q6. Shortcut key to paste copied text or image is:
a) Ctrl + C
b) Ctrl + V
c) Ctrl + X
d) Ctrl + P
Correct answer: b) Ctrl + V
Q7. To delete selected text or an image, you should:
a) Press Delete or Backspace
b) Press Ctrl + S
c) Press Ctrl + C
d) Press Ctrl + P
Correct answer: a) Press Delete or Backspace

--- PAGE 3 ---
Q8. To change the font, size, or color of text, you should use:
a) Home tab → Font group
b) Insert tab → Picture
c) Layout tab → Margins
d) Review tab → Spell Check
Correct answer: a) Home tab → Font group
Q9. To crop an image, you usually:
a) Select the image → Picture Tools → Crop
b) Select text → Font group
c) Insert → Table
d) File → Save
Correct answer: a) Select the image → Picture Tools → Crop
Q10. Adding and editing text and images helps to:
a) Make the document or slide more readable and visually
appealing
b) Save the document
c) Print the document
d) Align the paragraphs only
Correct answer: a) Make the document or slide more
readable and visually appealing
